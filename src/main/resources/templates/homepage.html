Welcome to the F1 Management System!

The purpose of this Website:

This Website is part of a course project for the 'Web Development' course at the 'Babes-Bolyai University' in Cluj-Napoca, Romania.
The project is a simple application for managing one or more F1 Teams. The application is divided into six main sections:

  • Teams
  • Drivers
  • Mechanics
  • Cars
  • Races
  • Participation's
Each section has its own page, where the user can view data or add, delete, edit data under the 'Teams' section.

Here is a brief summary of the application:

Homepage: A central page, where the application's description can be found.
Teams page: On this page new teams can be added, or older ones edited/deleted. Once a new team is added it can be managed using a passcode which must be provided on the first attempt to manage the team. Once it is added, there is no possibility to change or to recover the passcode yet, so it is recommended to save it somewhere. Under 'Team Management' drivers, mechanics, cars and participation's of a specific team can be managed.
It is strongly recommended for IDs to give a custom team-based id: ex. for team 1 --> id: 1x (where x is the wanted id of the entity/object)
Drivers page: This is a 'View only page' where the added drivers can be seen. Drivers can be managed only in the 'Team Management' section.
Mechanics page: This is a 'View only page' where the added mechanics can be seen. Mechanics can be managed only in the 'Team Management' section.
Cars page: This is a 'View only page' where the added cars can be seen. Cars can be managed only in the 'Team Management' section.
Races page: On this page new races can be added by the Race Manager via entering his passcode (only known by him).
Participation's page: This is a 'View only page' where the added participation can be seen. Participation's can be managed only in the 'Team Management' section.
